# AI Career Coach Dashboard

AI Career Coach Dashboard helps job seekers compare their resume or skills against a job description.

This beginner version works without a paid AI API. It uses simple keyword matching, polished UI, and browser LocalStorage so you can learn the core ideas before adding real AI.

## Features

- Paste resume or skills text
- Paste a job description
- Generate a match score
- See matched skills and missing skills
- Get a beginner learning plan
- Generate interview practice questions
- Track job applications
- Save tracker data in LocalStorage
- Export a text summary

## Tech Stack

- HTML
- CSS
- JavaScript
- LocalStorage
- Vercel-ready static hosting

## How To Run

Open `index.html` in your browser.

No install command is needed.

## How To Upload To GitHub

1. Open your GitHub repo: `https://github.com/kaneisha2412/ai`
2. Click **Add file**.
3. Click **Upload files**.
4. Upload the files listed in `GITHUB_UPLOAD_CHECKLIST.md`.
5. Click **Commit changes**.

## How To Deploy On Vercel

1. Go to `https://vercel.com`.
2. Sign in with GitHub.
3. Click **Add New Project**.
4. Import `kaneisha2412/ai`.
5. Keep the default settings.
6. Click **Deploy**.

Because this is a static HTML/CSS/JavaScript project, Vercel does not need a build command.
