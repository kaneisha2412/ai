# GitHub Upload Checklist

Target repo: `https://github.com/kaneisha2412/ai`

## Files To Upload

Upload these files into the root of the repo:

- `index.html`
- `styles.css`
- `app.js`
- `README.md`
- `DEPLOYMENT.md`
- `ROADMAP.md`
- `BACKLOG.md`
- `INTERVIEW_GUIDE.md`
- `GITHUB_UPLOAD_CHECKLIST.md`
- `PACKAGE_NOTES.md`
- `vercel.json`

## Browser Upload Steps

1. Open `https://github.com/kaneisha2412/ai`.
2. Click **Add file**.
3. Click **Upload files**.
4. Drag all files listed above into the upload area.
5. In the commit message box, type:

```text
Add AI Career Coach Dashboard starter project
```

6. Click **Commit changes**.

## After Uploading

Open the repo and check that these files appear:

- `index.html`
- `styles.css`
- `app.js`
- `README.md`

If they appear, the project is ready for Vercel.

## Vercel Settings

When importing into Vercel:

- Framework Preset: Other
- Build Command: leave empty
- Output Directory: leave empty
- Install Command: leave empty
