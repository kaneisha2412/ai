# Project Roadmap

This roadmap turns the AI Career Coach Dashboard into a stronger resume project step by step.

## Version 1: Static Dashboard

Status: Complete

What it proves:

- HTML structure
- CSS layout
- JavaScript DOM updates
- Responsive design
- LocalStorage persistence

Features:

- Resume and job description inputs
- Match score
- Matched and missing skills
- Learning plan
- Interview questions
- Application tracker
- Export summary

## Version 2: Better User Experience

Recommended next tasks:

- Add role presets such as Frontend Developer, Data Analyst, and Backend Developer.
- Add a progress chart for application status.
- Add edit and delete actions for individual tracker rows.
- Add a dark mode toggle.
- Improve keyboard accessibility.

## Version 3: React Upgrade

Recommended next tasks:

- Convert the project to React and Vite.
- Split the dashboard into components.
- Move analysis logic into utility functions.
- Add a cleaner state management pattern.

## Version 4: Real AI Integration

Recommended next tasks:

- Add a serverless API endpoint.
- Store the API key as a Vercel environment variable.
- Generate personalized resume feedback.
- Generate role-specific interview questions.
- Generate a 7-day learning plan.

## Version 5: Full Product Upgrade

Recommended next tasks:

- Add user login.
- Save applications to a database.
- Add charts and analytics.
- Add resume PDF upload.
- Add downloadable reports.
