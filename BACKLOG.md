# Beginner Backlog

Use this file as your learning checklist. Each item can become a GitHub issue.

## Easy

- [ ] Add three role preset buttons.
- [ ] Add a footer with your GitHub profile link.
- [ ] Add a project screenshot to the README.
- [ ] Add a delete button for each application row.
- [ ] Improve empty states with clearer messages.

## Medium

- [ ] Add an edit button for tracker rows.
- [ ] Add filters for application status.
- [ ] Add a simple application status chart.
- [ ] Add a dark mode toggle.
- [ ] Add more skills to the skill catalog.

## Hard

- [ ] Convert the app to React and Vite.
- [ ] Add a real AI API for resume feedback.
- [ ] Add login and database storage.
- [ ] Add resume PDF upload.
- [ ] Add downloadable reports.

## Good First Issue

Add role preset buttons:

- Frontend Developer
- Data Analyst
- Backend Developer

When a user clicks a preset, the app should fill the job description box with a sample job post.
