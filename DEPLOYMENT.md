# Beginner Deployment Guide

This guide gets the AI Career Coach Dashboard from your computer to GitHub and then to Vercel.

## Part 1: Put The Files On GitHub

1. Open `https://github.com/kaneisha2412/ai`.
2. Click **Add file**.
3. Click **Upload files**.
4. Upload these files:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `README.md`
   - `vercel.json`
5. Click **Commit changes**.

## Part 2: Deploy With Vercel

1. Open `https://vercel.com`.
2. Sign in with GitHub.
3. Click **Add New Project**.
4. Choose the GitHub repo `kaneisha2412/ai`.
5. Leave the build command empty.
6. Leave the output directory empty.
7. Click **Deploy**.

## Part 3: What To Put On Your Resume

Use this after the project is live:

```text
Built and deployed an AI-inspired career dashboard using HTML, CSS, and JavaScript to compare resumes against job descriptions, calculate match scores, identify missing skills, generate learning plans, and track job applications with LocalStorage.
```

## Part 4: Next Upgrade

After the first deploy works, upgrade the app in this order:

1. Convert it to React and Vite.
2. Add charts for application status.
3. Add a real AI API for resume feedback.
4. Add database storage.
5. Add login.
